#PayNowEatLater

Integration des Banners auf der eigenen Website

Um das Banner auf der eigenen Website zu integrieren genügt es die folgenden Script-Tags irgendwo im HTML auf der Seite einzufügen. 

<script type="text/javascript" src="{{ '/assets/javascripts/paynoweatlater.js' | absolute_url }}"></script>
<script type="text/javascript">
  paynoweatlater.addBanner('NAMEN_EINFÜGEN');
</script>

Damit die die richtige Gutscheinseite geladen wird, muss 'NAMEN_EINFÜGEN' auf den Namen in der Adresse der Gutscheinseite auf www.paynoweatlater.de angpasst werden.

Für Balz & Balz lautet die Adresse der Gutschein-Seite zum Beispiel https://www.paynoweatlater.de/at/balz-und-balz-hamburg/. Der einzufügende Name lautet also balz-und-balz-hamburg. Beim Einfügen dürfen die Anführungszeichen nicht vergessen werden, so dass die vollständige Zeile so aussieht: paynoweatlater.addBanner('balz-und-balz-hamburg');

Zum Ausprobieren ist das Banner auf dieser Seite wie im obigen Beispiel beschrieben konfiguriert.
