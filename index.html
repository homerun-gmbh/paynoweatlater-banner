#PayNowEatLater

Integration des Banners auf der eigenen Website

Um das Banner auf der eigenen Website zu integrieren, genügt es die folgenden Script-Tags irgendwo im HTML auf der Seite einzufügen: 

<script type="text/javascript" src="{{ '/assets/javascripts/paynoweatlater.js' | absolute_url }}"></script>
<script type="text/javascript">
  paynoweatlater.addBanner('GUTSCHEIN_URL');
</script>

Damit auf die eigene Gutscheinseite verwiesen wird, muss GUTSCHEIN_URL durch die entsprechende Adresse bei www.paynoweatlater.de ersetzt werden.

Beispiel

Für Balz und Balz zum Beispiel lautet die Adresse https://www.paynoweatlater.de/at/balz-und-balz-hamburg/. Diese Adresse muss anstelle von GUTSCHEIN_URL angegeben werden. Bitte vergesst beim Ändern nicht die Anführungszeichen, ohne diese funktioniert es nicht.

Für Balz und Balz sehen die Script-Tags nach der Änderung so aus: 

<script type="text/javascript" src="{{ '/assets/javascripts/paynoweatlater.js' | absolute_url }}"></script>
<script type="text/javascript">
  paynoweatlater.addBanner('https://www.paynoweatlater.de/at/balz-und-balz-hamburg/');
</script>

Zum Ausprobieren ist das Banner auf dieser Seite wie oben angegeben konfiguriert.
