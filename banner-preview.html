#PayNowEatLater

Banner-Vorschau zum einfacheren Editieren

Es ist einfacher das Banner in HTML zu editieren, es wird aber im JavaScript als String gebraucht. Ein Klick auf »Als Text« zeigt den Quelltext des Banners als Text an, so dass es einfach kopiert werden kann. Im JavaScript muss dieser dann »nur noch« eingefügt und entsprechend formatiert werden.

In seiner ursprünglichen Version ist das Banner so konfiguriert, dass es bei einer Bildschirmbreite kleiner 500px die volle Breite ausnutzt.
